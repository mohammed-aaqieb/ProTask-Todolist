# ProTask-Todolist
A professional Todo List Dashboard built with Flask & Vanilla JS. Features a premium "Dark Glass" UI, real-time task management, priority levels, and a Kanban-style workflow. Powered by SQLite & SQLAlchemy for robust performance.

## Features

- **Dashboard**: Responsive layout with sidebar navigation and real-time statistics.
- **Task Management**: Create, edit, and delete tasks with detailed descriptions and priority levels.
- **Status Workflow**: Track tasks through "Todo", "In Progress", and "Done" stages.
- **Modern UI**: Glassmorphism design with smooth animations and a dark theme.
- **Search**: Real-time filtering of tasks.

## Tech Stack

- **Backend**: Python, Flask, SQLAlchemy
- **Database**: SQLite
- **Frontend**: HTML5, CSS3, JavaScript

## Installation

1.  **Clone the repository**
    ```bash
    git clone (https://github.com/mohammed-aaqieb/ProTask-Todolist.git)
    cd "Todo list"
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Application**
    ```bash
    python app.py
    ```

4.  **Access the App**
    Open `http://127.0.0.1:5000` in your browser.
